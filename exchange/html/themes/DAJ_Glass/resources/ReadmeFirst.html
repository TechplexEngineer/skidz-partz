Installation
  1. Unzip the file (which you already have) and you should see 1 readme file and 2 folders - resources/ and html/.
  2. Upload all files IN the HTML folder to your nuke root. Nuke root is the location of your nuke site where admin.php, config.php, and mainfile.php are located. Do NOT upload any files from the resources folder to your nuke root. Those are not part of the installation files. Only upload the html folder's contents.
  3. Test the theme by selecting it in Your Account. Do Not select the theme in the admin preferences until you are sure it was installed correctly.

There is a theme matching "Recent Posts" block included with some themes as well. This block was designed to be used as a center block.
